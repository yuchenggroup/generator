<property> 元素

<property> 元素用于指定许多其他元素的属性。 每个支持属性元素的文档页面列出的不同值都是有效的。 此元素也可以用于将属性传递给您的任何自定义的代码生成器执行。

必选属性

属性 描述
name 属性的名称(区分大小写)。
value 属性的值(通常是不区分大小写)。

可选属性

无

子元素

无
